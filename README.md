## Install

1. Clone the repo
   ```sh
   git clone https://github.com/kuouu/smart-room-frontend.git
   ```
2. Install YARN packages
   ```sh
   yarn
   ```
4. Enter your URL in `/.env`
   ```
   REACT_APP_HOME_CONTROL_BACKEND='ENTER YOUR BACKEND URL'
   ```
5. Run the React application
   ```sh
   yarn start
   ```
